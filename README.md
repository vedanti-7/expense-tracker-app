#  Expense Tracker App

A Flutter expense tracker app that helps users manage expenses across multiple categories. Users can add and view expenses, categorize them, and swipe to dismiss items from the home screen if desired. Designed with a clean, responsive UI for effortless financial tracking and organization.

- Works smoothly across devices
